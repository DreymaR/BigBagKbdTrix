MacOS Platform Info

There is an implementation of the Big Bag for Karabinier Elements by CSGO from the Colemak Discord (also known as Amogus or Professor X-Wing). He's made a script to generate different variants. There's also a repo for just the Extend layer if that's what you want. This is very promising for the Mac crowd!

I hear good things about the KMonad and Kanata key remapping tools; these seem able to implement key mapping layers and Extend well across platforms. Maybe I'll get around to trying them out.

There is also an older implementation that I know of: Colemak Forum user mkborregaard implemented some of the Big Bag (Mac) for Karabinier (not maintained by me), but I'm told this tool doesn't work for MacOS Sierra onwards?

If anyone has good news about Apple compatible implementations of the BigBag or similar tricks, please let me know.
