MacOS Platform Info

Sadly, there isn't a lot of content for the Apple user here. Colemak Forum user mkborregaard implemented some of the Big Bag (Mac) for Karabinier (not maintained by me), but this tool doesn't work well anymore.

There is a newer implementation of the Big Bag Extend layer for Karabinier Elements by CSGO from the Colemak Discord (also known as Amogus or Professor X-Wing). He's made a script to generate different variants. This looks very promising for the Mac crowd!

I hear good things about the KMonad Keyboard Management Tool, and it seems able to implement layers and Extend well across platforms. Maybe I'll get around to trying it out. For now, if anyone has good news about Apple platform implementations of the BigBag or similar tricks, please let me know.
