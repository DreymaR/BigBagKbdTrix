MacOS Extend Info

There is an implementation of the Big Bag Extend layer for Karabinier Elements by CSGO from the Colemak Discord (also known as Amogus or Professor X-Wing). He has also made a script to generate different Big Bag variant combos. Have a look!


Mattmc3 made a Karabinier-Elements config generator for his keyboard tools in 2019. It looks cool, but I haven't tried it out myself.

Zhang0313, similarly, made a Karabinier .json for Extend in 2019, that should be compatible with MacOS Sierra.

Mkborregaard on the Colemak Forums implemented some of the Big Bag (Mac) for Karabinier back in 2015, but I'm told this tool doesn't work for MacOS Sierra onwards?

I don't know if or how the above are different from ProfXWing's implementation. All in all, I'd recommend checking that one out first, as its maker is still active at the Colemak Discord (under the name CSGO or something like that).
