Windows Extend Info

Extend is implemented for Windows in the Big Bag for (E)PKL repository on GitHub. See the Platforms section on the index page and links therein for more info.

I know of no good way to implement Extend for an installable (MSKLC) layout for Windows. My EPKL program, on the other hand, has the best Extend implementation available that I know of.

EPKL specific Extend info and settings are found on the main Layers page.
